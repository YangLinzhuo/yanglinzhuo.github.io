# 前言

时隔数个月，总算又重新敲打出这些文字。经过了几个月对毕业设计及其他相关事宜的处理之后，总算要踏上一段新的旅途了。现在也总算是在新城市安顿下来，再过段时间便也成为真正的社会人了。最近仍旧处在我自己设定的放假时间，通关了去年的新游戏《破晓传说》，周杰伦的新专辑也发售了，诸多事情，有感而发，遂有此文。

# 破晓传说

作为传说系列的最新作，破晓传说在成绩上是令人满意的，本作销量也超过了两百万，算是重新拯救了因为前作情热传说而濒临危险境地的这个 IP。综合来看，我个人对于本作是比较满意的。游戏的前半部分篇章在剧情的设计及节奏把控上都是非常不错的，场景设计也比较符合各个章节对应领将的属性。在战斗系统上，也非常流畅，个人玩起来的打击感非常不错，各种招式的动画效果也非常炫酷。

遗憾的是，后半部分的剧情内容明显节奏偏快，剧情信息密度过大，给人一种在疯狂弥补设定的感觉。后半部分的迷宫设计也明显没有之前的用心，大量的必须战斗对于人的耐心也是一种巨大的消耗。此外，BOSS 战斗几乎永久霸体的设定，加上针对玩家的仇恨锁定，使得我在后期难度较大的地图中的游玩过程明显没有前半部分那样的顺畅。

虽然如此，我个人还是比较推荐本作，尤其是在前半部分，在精美画面和人设的加持下，算是让我久违地体验到了 RPG 游戏带来的乐趣。在第一部分结束后的 OP2 也让人无比期待后续的剧情，可惜后半部分确实让人有些失望，但是瑕不掩瑜，最后也是 HE，值得一玩。

绚香演唱的 OP2 「Hello,Again～昔からある場所～」配合上飞碟社的动画，让我体验到了久违的沉浸感。雨景很好地契合了前一章节场景的特点，也渲染了一种怅惘的氛围，和歌词的部分也很好地结合了起来。绚香略带厚重的女声也让我不禁沉浸在其场景之后，之后几乎每次进游戏都要完整听一遍。这首歌其实 1995 年的一首歌曲，原曲明显节奏更加明快。经过改编后的现有版本和游戏画面契合度非常高，是非常不错的翻唱。

# 周杰伦

说起怀旧，马上就是周杰伦专辑《八度空间》20 周年了，而新专辑也已经发行。每当周杰伦发行新歌的时候，总有人会说江郎才尽了，不如以前了之类的。我不懂专业的音乐知识，无法从专业的角度进行评价。而在 20 年前，《八度空间》这张专辑在金曲奖上颗粒无收，就像周杰伦后来在《外婆》中唱的那样，只能在镜头前傻笑。

然而，在如今看来，《八度空间》的每首歌曲都有值得品鉴的地方。也许是回忆的滤镜使得人们对于过往的事物有着很高的评价，使得人们对于当前的新事物总是在评价上过于保守。就好像很多人发现，周杰伦的新歌过一段时间听之后仿佛又有新鲜的感觉。固然，和以前相比，周杰伦的心境已经发生了明显的变化，如今的他已经是华语乐坛的 “帝王”，创作的那种迫切感当然无法与年轻时候相比，但是如果我们拿这样的说辞去苛求他未免也有些过界了。音乐本就是一种众口难调的艺术，个人的阅历也会在我们聆听的时候带给我们不一样的感受。或许，我们可以过段时间再重新品尝一下新专辑的滋味。如果仍然觉得不合口味，那么适时怀旧又有何不可呢？

最后，推荐 B 站品鉴《八度空间》的这段视频作为结尾。
